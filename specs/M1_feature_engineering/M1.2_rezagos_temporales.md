---

## Agente

**Rol**: Eres un cientifico de datos especializado en series temporales. Debes
calcular rezagos y tendencias sin usar informacion del año objetivo ni del futuro.

**Mentalidad**: Piensa en fuga de informacion. Cada feature rezagada debe basarse en
anos estrictamente anteriores al target y respetar el split temporal del proyecto.

**Restricciones**:

- No usar `inplace=True`
- No usar datos del año t ni posteriores para construir lags
- Respetar el split: TRAIN_HASTA=2021, VAL_AÑO=2022, TEST_DESDE=2023

---

## Objetivo

Calcular rezagos de rendimiento y area sembrada por `codigo_dane + cultivo`, junto con
promedios y tendencia de 3 años, sin fuga de informacion.

---

## Inputs

| Input | Tipo           | Origen         |
| ----- | -------------- | -------------- |
| `df`  | `pd.DataFrame` | Output de M1.1 |

Columnas requeridas: `codigo_dane`, `cultivo`, `año`, `rendimiento`, `area_sembrada`.

---

## Output

DataFrame con nuevas columnas:
`rendimiento_t1`, `rendimiento_prom3a`, `tendencia_rend_3a`, `area_sembrada_t1`.

---

## Logica de transformacion

```
1. Ordenar por [codigo_dane, cultivo, año]
2. Agrupar por (codigo_dane, cultivo)
3. rendimiento_t1 = shift(1) de rendimiento
4. rendimiento_prom3a = promedio de rendimiento en t-1..t-3
   - min_periods=2 (si hay <2 valores no-NaN → NaN)
5. tendencia_rend_3a = pendiente de regresion lineal en t-1..t-3
   - requiere 3 valores no-NaN; si no, NaN
6. area_sembrada_t1 = shift(1) de area_sembrada
7. Restringir fuentes segun split temporal:
   - Para año t <= 2021: usar solo años < t
   - Para año t == 2022: usar solo años <= 2021
   - Para año t >= 2023: usar solo años <= 2022
   Si la fuente mas reciente excede el limite permitido → forzar NaN
```

## Criterios de aceptacion

- [ ] `rendimiento_t1` usa solo t-1; si no existe o es NaN, el lag es NaN
- [ ] `rendimiento_prom3a` usa t-1..t-3 y requiere >=2 valores no-NaN
- [ ] `tendencia_rend_3a` requiere 3 valores no-NaN; si no, NaN
- [ ] `area_sembrada_t1` usa solo t-1; si no existe o es NaN, el lag es NaN
- [ ] No se usan datos del año t ni posteriores
- [ ] Para 2022, solo se usa historia <= 2021
- [ ] Para 2023+ solo se usa historia <= 2022
