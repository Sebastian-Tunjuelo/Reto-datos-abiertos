## Logica de transformacion

### 1) `_validate_input(df)`

```
1. Si df esta vacio → ValueError("[M1] tabla_maestra esta vacia")
2. Verificar columnas: codigo_dane, cultivo, año
   - Si faltan → ValueError("[M1] Columnas faltantes: ...")
3. Filtrar cultivos fuera de CULTIVOS_MVP
   - Loggear: "[M1] {n} filas descartadas — cultivos no reconocidos: {valores}"
4. Normalizar codigo_dane con normalize_dane_code()
   - Si retorna None o falla → descartar fila y loggear warning
5. Filtrar años fuera de 2007–2024
   - Loggear: "[M1] {n} filas descartadas — año fuera de rango 2007–2024"
6. Si todas las filas se descartan por año → ValueError
```

### 2) `_apply_climate_ranges(df)`

```
1. Para cada variable climatica en RANGOS_CLIMATICOS:
   - Si la columna no existe → crear columna NaN y warning
   - Si el valor esta fuera de rango → reemplazar por NaN y warning por fila
2. Si las 6 variables son NaN para (codigo_dane, año) → warning
3. Propagar NaN ya existentes sin modificar
```
