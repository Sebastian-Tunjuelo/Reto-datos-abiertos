## Lógica de agregación anual

### Paso 1 — Agrupar por año
```python
# Para cada columna numérica: promedio de los meses disponibles ese año
anual = mensual.groupby('año').agg(
    indice_total   = ('indice_total',   'mean'),
    fertilizantes  = ('fertilizantes',  'mean'),
    plaguicidas    = ('plaguicidas',    'mean'),
    urea           = ('urea',           'mean'),
    dap            = ('dap',            'mean'),
    kcl            = ('kcl',            'mean'),
    n_meses        = ('mes',            'count'),  # cuántos meses tienen datos
).reset_index()
```

> `n_meses` cuenta los meses con al menos un valor no nulo en `mes`. Un año con `n_meses < 6` se considera incompleto — loggear warning pero no eliminar.

### Paso 2 — Percentiles históricos con ventana deslizante

Para cada año `t`, el percentil se calcula usando **solo los años anteriores** (`año < t`). Esto evita fuga de información: cuando el modelo predice para 2022, solo conoce los percentiles calculados con datos hasta 2021.

```
Para cada año t en el DataFrame anual:
    historico = anual[anual['año'] < t]
    Si len(historico) < 3:
        pct_fertilizantes[t] = NaN  # insuficiente historia
        pct_indice_total[t]  = NaN
    Sino:
        pct_fertilizantes[t] = percentileofscore(historico['fertilizantes'].dropna(),
                                                  anual.loc[t, 'fertilizantes'],
                                                  kind='rank') / 100
        pct_indice_total[t]  = percentileofscore(historico['indice_total'].dropna(),
                                                  anual.loc[t, 'indice_total'],
                                                  kind='rank') / 100
```

Usar `scipy.stats.percentileofscore(a, score, kind='rank')` — retorna valor 0–100, dividir por 100 para obtener 0.0–1.0.

> **Umbral mínimo de historia**: Si hay menos de 3 años anteriores con datos, el percentil es `NaN`. Los primeros 2 años del dataset (típicamente 2007 y 2008) tendrán `pct_*` nulos — esto es correcto y esperado.

### Paso 3 — Señal de riesgo económico

```python
def calcular_señal_riesgo(pct_fertilizantes: float | None) -> str:
    """
    Clasifica el riesgo económico por costo de insumos.
    Basado en el percentil histórico del sub-índice de fertilizantes.
    """
    if pd.isna(pct_fertilizantes):
        return 'Medio'  # valor por defecto cuando no hay historia suficiente
    if pct_fertilizantes >= 0.75:
        return 'Alto'
    if pct_fertilizantes >= 0.40:
        return 'Medio'
    return 'Bajo'
```

| Percentil fertilizantes | Señal de riesgo | Interpretación |
|------------------------|-----------------|----------------|
| ≥ 0.75 (p75) | `'Alto'` | Año muy caro en insumos — riesgo de reducción de área sembrada |
| 0.40 – 0.74 | `'Medio'` | Año con costos moderados |
| < 0.40 | `'Bajo'` | Año barato en insumos — condiciones favorables |
| `NaN` | `'Medio'` | Sin historia suficiente — asumir riesgo neutro |

> La señal se basa en `fertilizantes` (no en `indice_total`) porque los fertilizantes son el componente de mayor impacto en el costo de producción de los 3 cultivos del MVP.

## Notas de implementación

### `scipy.stats.percentileofscore`
Importar como:
```python
from scipy.stats import percentileofscore
```
`kind='rank'` es el método más robusto para series pequeñas — maneja empates promediando los rangos. Alternativa si scipy no está disponible: `np.searchsorted(np.sort(historico), valor) / len(historico)`.

### Años con datos incompletos
Un año con `n_meses < 12` puede tener promedios sesgados (ej. si solo hay datos de enero–junio, el promedio no representa el año completo). Loggear un warning para estos casos pero no eliminarlos — el modelo puede aprender a usar `n_meses` como feature de confianza.

### Año actual (datos parciales)
Si el pipeline se ejecuta en 2025 y el dataset tiene datos hasta marzo 2025, el año 2025 tendrá `n_meses = 3`. Esto es correcto — el modelo usará el promedio de los meses disponibles.

### Orden de columnas en el output
El DataFrame anual debe tener las columnas exactamente en este orden:
```python
['año', 'indice_total', 'fertilizantes', 'plaguicidas', 'urea', 'dap', 'kcl',
 'n_meses', 'pct_fertilizantes', 'pct_indice_total', 'señal_riesgo']
```
