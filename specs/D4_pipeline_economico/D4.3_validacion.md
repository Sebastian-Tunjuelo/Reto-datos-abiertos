## Agente

**Rol**: Eres un ingeniero de QA de datos. Tu trabajo es escribir un script de validación independiente que actúe como árbitro objetivo del pipeline económico. No tienes acceso al código de producción — solo lees los archivos Parquet resultantes y verificas que cumplen el contrato de datos.

**Mentalidad**: Piensa como un auditor, no como el autor del pipeline. Los datos de agroinsumos son una serie temporal sin dimensión geográfica, por lo que los errores aquí son de integridad temporal: ¿están todos los años? ¿los percentiles son consistentes? ¿la señal de riesgo tiene los valores correctos? ¿los percentiles de los primeros años son NaN como se espera? Cada aserción debe tener un mensaje de error que explique exactamente qué falló. Prefiere aserciones granulares sobre aserciones genéricas.

**Restricciones**:
- No importar ni llamar funciones del módulo `modules/economic/` — solo leer Parquets
- No modificar los archivos Parquet bajo ninguna circunstancia
- Cada aserción debe tener un mensaje de error con el prefijo `❌` y descripción del problema
- El script debe poder ejecutarse de forma aislada sin depender del estado del proceso anterior
- Si una aserción falla, el script debe detenerse inmediatamente con `AssertionError` — no continuar acumulando errores

## Checklist de criterios de aceptación

### Estructura de archivos
- [ ] `data/agroinsumos_mensual.parquet` existe
- [ ] `data/agroinsumos.parquet` existe

### Esquema mensual — `agroinsumos_mensual.parquet`
- [ ] Las 9 columnas del esquema mensual están presentes: `fecha, año, mes, indice_total, fertilizantes, plaguicidas, urea, dap, kcl`
- [ ] `fecha` es `object` (string) con formato `'YYYY-MM'` en todos los registros
- [ ] `año` es `int64` o `Int64`; valores en rango 2007–2024
- [ ] `mes` es `int64` o `Int64`; valores en rango 1–12
- [ ] `indice_total`, `fertilizantes`, `plaguicidas`, `urea`, `dap`, `kcl` son `float64`
- [ ] No hay duplicados por `(año, mes)`
- [ ] El DataFrame está ordenado por `(año, mes)` ascendente

### Esquema anual — `agroinsumos.parquet`
- [ ] Las 11 columnas del esquema anual están presentes: `año, indice_total, fertilizantes, plaguicidas, urea, dap, kcl, n_meses, pct_fertilizantes, pct_indice_total, señal_riesgo`
- [ ] `año` es `int64`; valores en rango 2007–2024
- [ ] `n_meses` es `int64`; valores en rango 1–12
- [ ] `indice_total`, `fertilizantes`, `plaguicidas`, `urea`, `dap`, `kcl` son `float64`
- [ ] `pct_fertilizantes`, `pct_indice_total` son `float64`; valores en [0.0, 1.0] o `NaN`
- [ ] `señal_riesgo` contiene solo `{'Bajo', 'Medio', 'Alto'}`; sin nulos
- [ ] No hay duplicados por `año`
- [ ] El DataFrame está ordenado por `año` ascendente

### Integridad temporal
- [x] El archivo mensual tiene al menos 5 años de cobertura (≥ 60 registros)
- [x] El archivo anual tiene al menos 6 años de cobertura
- [ ] Los primeros 2 años del archivo anual tienen `pct_fertilizantes = NaN` y `pct_indice_total = NaN`
- [ ] Los años con suficiente historia (año > año_min + 2) tienen `pct_fertilizantes` no nulo

### Consistencia entre archivos
- [ ] El conjunto de años en `agroinsumos.parquet` es subconjunto de los años en `agroinsumos_mensual.parquet`
- [ ] Para cada año, `n_meses` en el anual coincide con el conteo de registros en el mensual

### Integridad de percentiles
- [ ] `pct_fertilizantes` y `pct_indice_total` están en [0.0, 1.0] donde no son `NaN`
- [ ] Los percentiles son monótonamente consistentes: si `fertilizantes[t1] < fertilizantes[t2]`, entonces `pct_fertilizantes[t1] <= pct_fertilizantes[t2]` (verificar al menos 3 pares)

## Script de validación
Guardar como `specs/D4_pipeline_economico/validate_d4.py`:

```python
"""
Script de validación para el pipeline D4 — Económico (Agroinsumos).
Ejecutar después de modules.economic.ingestion.run_pipeline()
"""
import sys
from pathlib import Path

import pandas as pd
import numpy as np

ROOT = Path(__file__).parent.parent.parent
sys.path.insert(0, str(ROOT))

from shared.config import DATA_DIR

COLUMNAS_MENSUAL = [
    "fecha", "año", "mes",
    "indice_total", "fertilizantes", "plaguicidas", "urea", "dap", "kcl",
]
COLUMNAS_ANUAL = [
    "año", "indice_total", "fertilizantes", "plaguicidas", "urea", "dap", "kcl",
    "n_meses", "pct_fertilizantes", "pct_indice_total", "señal_riesgo",
]
SEÑALES_VALIDAS = {"Bajo", "Medio", "Alto"}


def run_validations(data_dir: Path = DATA_DIR) -> None:
    """Ejecuta todas las validaciones. Lanza AssertionError si alguna falla."""

    # --- Existencia de archivos ---
    for nombre in ["agroinsumos_mensual", "agroinsumos"]:
        path = data_dir / f"{nombre}.parquet"
        assert path.exists(), f"❌ Archivo no encontrado: {path}"

    mensual = pd.read_parquet(data_dir / "agroinsumos_mensual.parquet")
    anual   = pd.read_parquet(data_dir / "agroinsumos.parquet")

    # --- Esquema mensual ---
    faltantes_m = set(COLUMNAS_MENSUAL) - set(mensual.columns)
    assert not faltantes_m, f"❌ agroinsumos_mensual: columnas faltantes: {faltantes_m}"

    # Formato de fecha 'YYYY-MM'
    patron_fecha = mensual["fecha"].str.match(r'^\d{4}-\d{2}$')
    assert patron_fecha.all(), (
        f"❌ agroinsumos_mensual: {(~patron_fecha).sum()} fechas con formato incorrecto"
    )

    # Tipos
    assert mensual["año"].dtype in ("int64", "Int64"), (
        f"❌ agroinsumos_mensual.año no es int64/Int64: {mensual['año'].dtype}"
    )
    assert mensual["mes"].dtype in ("int64", "Int64"), (
        f"❌ agroinsumos_mensual.mes no es int64/Int64: {mensual['mes'].dtype}"
    )
    for col in ["indice_total", "fertilizantes", "plaguicidas", "urea", "dap", "kcl"]:
        assert mensual[col].dtype == "float64", (
            f"❌ agroinsumos_mensual.{col} no es float64: {mensual[col].dtype}"
        )

    # Rangos
    años_m = mensual["año"].dropna()
    assert años_m.between(2007, 2024).all(), (
        f"❌ agroinsumos_mensual: años fuera de rango 2007-2024: "
        f"{años_m[~años_m.between(2007, 2024)].unique().tolist()}"
    )
    meses_m = mensual["mes"].dropna()
    assert meses_m.between(1, 12).all(), (
        f"❌ agroinsumos_mensual: meses fuera de rango 1-12: "
        f"{meses_m[~meses_m.between(1, 12)].unique().tolist()}"
    )

    # Duplicados
    dups_m = mensual.duplicated(["año", "mes"]).sum()
    assert dups_m == 0, f"❌ agroinsumos_mensual: {dups_m} duplicados por (año, mes)"

    # Orden
    mensual_sorted = mensual.sort_values(["año", "mes"])
    assert mensual.reset_index(drop=True).equals(mensual_sorted.reset_index(drop=True)), (
        "❌ agroinsumos_mensual: no está ordenado por (año, mes)"
    )

    # Cobertura mínima
    assert len(mensual) >= 144, (
        f"❌ agroinsumos_mensual: solo {len(mensual)} registros — se esperan ≥ 144 (12 años)"
    )

    # --- Esquema anual ---
    faltantes_a = set(COLUMNAS_ANUAL) - set(anual.columns)
    assert not faltantes_a, f"❌ agroinsumos: columnas faltantes: {faltantes_a}"

    # Tipos
    assert anual["año"].dtype == "int64", (
        f"❌ agroinsumos.año no es int64: {anual['año'].dtype}"
    )
    assert anual["n_meses"].dtype == "int64", (
        f"❌ agroinsumos.n_meses no es int64: {anual['n_meses'].dtype}"
    )
    for col in ["indice_total", "fertilizantes", "plaguicidas", "urea", "dap", "kcl",
                "pct_fertilizantes", "pct_indice_total"]:
        assert anual[col].dtype == "float64", (
            f"❌ agroinsumos.{col} no es float64: {anual[col].dtype}"
        )

    # Rangos
    años_a = anual["año"]
    assert años_a.between(2007, 2024).all(), (
        f"❌ agroinsumos: años fuera de rango 2007-2024: "
        f"{años_a[~años_a.between(2007, 2024)].tolist()}"
    )
    assert anual["n_meses"].between(1, 12).all(), (
        f"❌ agroinsumos: n_meses fuera de rango 1-12"
    )

    # Percentiles en [0, 1]
    for col in ["pct_fertilizantes", "pct_indice_total"]:
        vals = anual[col].dropna()
        fuera = vals[(vals < 0) | (vals > 1)]
        assert fuera.empty, f"❌ agroinsumos.{col}: valores fuera de [0.0, 1.0]: {fuera.tolist()}"

    # señal_riesgo
    señales_reales = set(anual["señal_riesgo"].unique())
    invalidas = señales_reales - SEÑALES_VALIDAS
    assert not invalidas, f"❌ agroinsumos.señal_riesgo: valores inválidos: {invalidas}"
    assert anual["señal_riesgo"].notna().all(), "❌ agroinsumos.señal_riesgo: hay nulos"

    # Duplicados
    dups_a = anual.duplicated("año").sum()
    assert dups_a == 0, f"❌ agroinsumos: {dups_a} duplicados por año"

    # Orden
    assert (anual["año"].diff().dropna() > 0).all(), (
        "❌ agroinsumos: no está ordenado por año ascendente"
    )

    # Cobertura mínima
    assert anual["año"].nunique() >= 10, (
        f"❌ agroinsumos: solo {anual['año'].nunique()} años — se esperan ≥ 10"
    )

    # --- Integridad temporal: primeros 2 años con NaN en percentiles ---
    año_min = anual["año"].min()
    primeros_2 = anual[anual["año"] <= año_min + 1]
    assert primeros_2["pct_fertilizantes"].isna().all(), (
        f"❌ agroinsumos: los primeros 2 años deberían tener pct_fertilizantes=NaN "
        f"(años {año_min} y {año_min+1})"
    )
    assert primeros_2["pct_indice_total"].isna().all(), (
        f"❌ agroinsumos: los primeros 2 años deberían tener pct_indice_total=NaN"
    )

    # Años con historia suficiente tienen percentiles no nulos
    años_con_historia = anual[anual["año"] > año_min + 2]
    nulos_pct = años_con_historia["pct_fertilizantes"].isna().sum()
    assert nulos_pct == 0, (
        f"❌ agroinsumos: {nulos_pct} años con historia suficiente tienen pct_fertilizantes=NaN"
    )

    # --- Consistencia entre archivos ---
    años_mensual = set(mensual["año"].dropna().unique())
    años_anual   = set(anual["año"].unique())
    extra = años_anual - años_mensual
    assert not extra, (
        f"❌ agroinsumos: años en anual que no están en mensual: {extra}"
    )

    # n_meses coincide con conteo en mensual
    conteo_mensual = mensual.groupby("año")["mes"].count().rename("n_meses_real")
    merged = anual.set_index("año").join(conteo_mensual, how="left")
    discrepancias = merged[merged["n_meses"] != merged["n_meses_real"]]
    assert discrepancias.empty, (
        f"❌ agroinsumos: n_meses no coincide con conteo mensual en años: "
        f"{discrepancias.index.tolist()}"
    )

    # --- Consistencia de percentiles (verificar monotonía en 3 pares) ---
    anual_sorted = anual.sort_values("fertilizantes").dropna(subset=["fertilizantes", "pct_fertilizantes"])
    if len(anual_sorted) >= 3:
        pct_sorted = anual_sorted["pct_fertilizantes"].values
        # Los percentiles deben ser no decrecientes cuando se ordena por el valor
        assert (np.diff(pct_sorted) >= -0.01).all(), (
            "❌ agroinsumos: pct_fertilizantes no es monótonamente consistente con fertilizantes"
        )

    # --- Resumen si todo pasó ---
    print("✅ Todos los criterios de aceptación pasaron")
    print("\nResumen:")
    print(f"  agroinsumos_mensual: {len(mensual):>5} registros | "
          f"años {int(mensual['año'].min())}–{int(mensual['año'].max())}")
    print(f"  agroinsumos anual:   {len(anual):>5} años     | "
          f"años {int(anual['año'].min())}–{int(anual['año'].max())}")
    print(f"\n  n_meses promedio: {anual['n_meses'].mean():.1f}")
    print(f"  Años con datos incompletos (n_meses < 12): "
          f"{(anual['n_meses'] < 12).sum()}")
    print(f"\n  Distribución señal_riesgo:")
    for señal in ["Bajo", "Medio", "Alto"]:
        n = (anual["señal_riesgo"] == señal).sum()
        print(f"    {señal:5s}: {n:>3} años ({n/len(anual):.0%})")
    print(f"\n  indice_total  — min: {anual['indice_total'].min():.1f} | "
          f"max: {anual['indice_total'].max():.1f} | "
          f"media: {anual['indice_total'].mean():.1f}")
    print(f"  fertilizantes — min: {anual['fertilizantes'].min():.1f} | "
          f"max: {anual['fertilizantes'].max():.1f} | "
          f"media: {anual['fertilizantes'].mean():.1f}")
    print(f"\n  Percentiles calculados para {anual['pct_fertilizantes'].notna().sum()} "
          f"de {len(anual)} años")


if __name__ == "__main__":
    run_validations()
```
