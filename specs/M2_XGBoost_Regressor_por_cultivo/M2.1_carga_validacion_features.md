## Objetivo

---

## Logica de validacion y preprocesamiento

```
1. Verificar que el archivo existe
   - Si no existe -> FileNotFoundError("M2.1: falta feature_matrix.parquet")
2. Cargar con pd.read_parquet(path)
3. Verificar columnas requeridas
   - Si faltan -> ValueError("M2.1: columnas faltantes: ...")
4. Filtrar a CULTIVOS_MVP y loggear descartes
5. Normalizar codigo_dane con normalize_dane_code()
   - Filas invalidas -> descartar con warning
6. Filtrar anios fuera de 2007-2024
   - Loggear numero de filas descartadas
7. Convertir columnas numericas con pd.to_numeric(..., errors="coerce")
8. Mapear señal_riesgo_economico (case-insensitive):
   bajo -> 0, medio -> 1, alto -> 2
   - Valores no mapeados -> NaN + warning
9. Eliminar filas con target_rendimiento NaN
10. Definir feature_cols:
    - Incluir columnas numericas y año
    - Excluir identificadores (codigo_dane, municipio, departamento, cultivo)
    - Excluir target_rendimiento
11. Retornar df limpio + feature_cols + target_col
```

---

## Output

- DataFrame validado y listo para entrenamiento
- `feature_cols` (lista ordenada de features)
- `target_col = "target_rendimiento"`

## Manejo de errores

| Situacion                        | Comportamiento                              |
| -------------------------------- | ------------------------------------------- |
| Archivo no existe                | `FileNotFoundError` con mensaje descriptivo |
| Columnas faltantes               | `ValueError` listando columnas faltantes    |
| Todas las filas filtradas        | `ValueError("M2.1: no hay filas validas")`  |
| Valores no mapeados en señal eco | Warning y reemplazo por NaN                 |
