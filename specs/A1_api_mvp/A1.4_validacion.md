## Agente

**Rol**: Eres un ingeniero de QA y MLOps encargado de certificar que la API del MVP expone el contrato esperado y que su comportamiento observable coincide con la spec de A1.

**Mentalidad**: Piensa como un auditor externo. No verificas cómo está implementada la lógica interna, sino que el contrato HTTP, los esquemas de respuesta y los errores observables sean correctos. Si un endpoint responde bien pero con una forma distinta a la acordada, la validación debe fallar.

**Restricciones**:

- No llamar funciones privadas de `modules/` para validar el contrato
- No modificar archivos de datos ni modelos
- Cada aserción debe tener un mensaje descriptivo con prefijo `❌`
- Si una aserción falla, el script debe detenerse inmediatamente con `AssertionError`
- El script debe poder ejecutarse de forma aislada

---

## Objetivo

Verificar que los tres endpoints del MVP funcionan como contrato público:

1. `GET /municipios`
2. `GET /cultivos/{municipio}`
3. `POST /predecir`

Además, comprobar que el contrato de error y la normalización de entradas se mantienen consistentes.

## Checklist de criterios de aceptación

### Disponibilidad básica

- [ ] `orchestrator.main.app` es importable
- [ ] `GET /openapi.json` responde y contiene los 3 endpoints del MVP

### `GET /municipios`

- [ ] Responde `200 OK`
- [ ] Devuelve exactamente 15 registros
- [ ] Cada registro tiene exactamente `codigo_dane`, `municipio` y `departamento`
- [ ] `codigo_dane` tiene 5 dígitos en todos los registros
- [ ] El orden coincide con `MVP_CODIGOS`
- [ ] No hay duplicados por `codigo_dane`
- [ ] `municipio` y `departamento` coinciden con `DANE_TO_NAME` y `DANE_TO_DEPT`

### `GET /cultivos/{municipio}`

- [ ] Para un municipio válido del MVP responde `200 OK`
- [ ] La respuesta contiene `codigo_dane`, `municipio`, `departamento` y `cultivos`
- [ ] `cultivos` es una lista no vacía
- [ ] Todos los cultivos devueltos pertenecen a `CULTIVOS_MVP`
- [ ] La lista de cultivos no tiene duplicados
- [ ] La lista respeta el orden canónico de `CULTIVOS_MVP`
- [ ] La consulta por nombre y por código DANE devuelve la misma entidad normalizada
- [ ] Un municipio inválido responde `404`

### `POST /predecir`

- [ ] Para una combinación válida responde `200 OK`
- [ ] La respuesta contiene exactamente `codigo_dane`, `municipio`, `departamento`, `cultivo`, `año`, `rendimiento_esperado`, `prob_riesgo_alto` y `etiqueta_riesgo`
- [ ] `codigo_dane` tiene 5 dígitos
- [ ] `cultivo` coincide con el solicitado
- [ ] `año` coincide con el año enviado en la request
- [ ] `rendimiento_esperado` es numérico y finito
- [ ] `prob_riesgo_alto` es numérico y está en el rango `[0, 1]`
- [ ] `etiqueta_riesgo` pertenece al conjunto `{Bajo, Medio, Alto}`
- [ ] Un cultivo inválido responde `422`
- [ ] Un año no válido o no futuro responde `422`
- [ ] Un municipio inválido responde `404`

## Script de validación

Guardar como `specs/A1_api_mvp/validate_a1.py`:

```python
"""
Script de validación para la API A1 del MVP.
Ejecutar después de exponer los endpoints /municipios, /cultivos/{municipio} y /predecir.
"""
from __future__ import annotations

import math
import sys
from pathlib import Path
from urllib.parse import quote

import pandas as pd
from fastapi.testclient import TestClient

ROOT = Path(__file__).parent.parent.parent
sys.path.insert(0, str(ROOT))

from shared.config import CULTIVOS_MVP, DATA_DIR
from shared.dane_codes import DANE_TO_DEPT, DANE_TO_NAME, MVP_CODIGOS


def _assert_keys_exact(payload: dict, expected: list[str], label: str) -> None:
    assert set(payload.keys()) == set(expected), (
        f"❌ {label}: claves inesperadas. Esperadas={expected}, recibidas={list(payload.keys())}"
    )


def _pick_first_valid_case(client: TestClient) -> tuple[str, str, str, int]:
    municipios = client.get("/municipios")
    assert municipios.status_code == 200, f"❌ /municipios respondió {municipios.status_code}"
    data = municipios.json()
    assert len(data) > 0, "❌ /municipios no devolvió registros"

    municipio_item = data[0]
    codigo_dane = municipio_item["codigo_dane"]
    municipio = municipio_item["municipio"]

    cultivos = client.get(f"/cultivos/{quote(municipio, safe='')}")
    assert cultivos.status_code == 200, (
        f"❌ /cultivos/{municipio} respondió {cultivos.status_code}"
    )
    cultivos_data = cultivos.json()
    assert cultivos_data["cultivos"], f"❌ {municipio} no tiene cultivos disponibles"

    feature_matrix = pd.read_parquet(DATA_DIR / "feature_matrix.parquet")
    max_year = int(pd.to_numeric(feature_matrix["año"], errors="coerce").max())
    future_year = max_year + 1

    return codigo_dane, municipio, cultivos_data["cultivos"][0], future_year


def run_validations() -> None:
    """Ejecuta todas las validaciones. Lanza AssertionError si alguna falla."""
    feature_path = DATA_DIR / "feature_matrix.parquet"
    assert feature_path.exists(), f"❌ Archivo no encontrado: {feature_path}"

    model_paths = [
        ROOT / "models" / "xgb_regressor_cafe.pkl",
        ROOT / "models" / "xgb_regressor_cacao.pkl",
        ROOT / "models" / "xgb_regressor_maiz.pkl",
        ROOT / "models" / "xgb_classifier_cafe.pkl",
        ROOT / "models" / "xgb_classifier_cacao.pkl",
        ROOT / "models" / "xgb_classifier_maiz.pkl",
        ROOT / "models" / "xgb_regressor_cafe_meta.json",
        ROOT / "models" / "xgb_regressor_cacao_meta.json",
        ROOT / "models" / "xgb_regressor_maiz_meta.json",
        ROOT / "models" / "xgb_classifier_cafe_meta.json",
        ROOT / "models" / "xgb_classifier_cacao_meta.json",
        ROOT / "models" / "xgb_classifier_maiz_meta.json",
    ]
    for path in model_paths:
        assert path.exists(), f"❌ Archivo no encontrado: {path}"

    from orchestrator.main import app

    client = TestClient(app)

    # --- OpenAPI y rutas ---
    openapi = client.get("/openapi.json")
    assert openapi.status_code == 200, f"❌ /openapi.json respondió {openapi.status_code}"
    openapi_data = openapi.json()
    paths = openapi_data.get("paths", {})
    for route in ["/municipios", "/cultivos/{municipio}", "/predecir"]:
        assert route in paths, f"❌ Falta la ruta en OpenAPI: {route}"

    # --- GET /municipios ---
    municipios_resp = client.get("/municipios")
    assert municipios_resp.status_code == 200, (
        f"❌ /municipios respondió {municipios_resp.status_code}"
    )
    municipios = municipios_resp.json()
    assert len(municipios) == 15, f"❌ /municipios devolvió {len(municipios)} registros"

    expected_codes = MVP_CODIGOS
    actual_codes = [item["codigo_dane"] for item in municipios]
    assert actual_codes == expected_codes, (
        f"❌ Orden o códigos incorrectos en /municipios. Esperado={expected_codes}, recibido={actual_codes}"
    )

    for item in municipios:
        _assert_keys_exact(item, ["codigo_dane", "municipio", "departamento"], "MunicipioItem")
        assert len(item["codigo_dane"]) == 5, f"❌ codigo_dane inválido: {item['codigo_dane']}"
        assert DANE_TO_NAME[item["codigo_dane"]] == item["municipio"], (
            f"❌ municipio no coincide para {item['codigo_dane']}"
        )
        assert DANE_TO_DEPT[item["codigo_dane"]] == item["departamento"], (
            f"❌ departamento no coincide para {item['codigo_dane']}"
        )

    assert len({item["codigo_dane"] for item in municipios}) == 15, "❌ Hay códigos DANE duplicados"

    # --- GET /cultivos/{municipio} ---
    codigo_dane, municipio, cultivo_valido, future_year = _pick_first_valid_case(client)

    by_name = client.get(f"/cultivos/{quote(municipio, safe='')}")
    assert by_name.status_code == 200, f"❌ /cultivos/{municipio} respondió {by_name.status_code}"
    by_code = client.get(f"/cultivos/{codigo_dane}")
    assert by_code.status_code == 200, f"❌ /cultivos/{codigo_dane} respondió {by_code.status_code}"

    payload_name = by_name.json()
    payload_code = by_code.json()
    assert payload_name == payload_code, "❌ La consulta por nombre y por código no devuelve la misma entidad"
    _assert_keys_exact(payload_name, ["codigo_dane", "municipio", "departamento", "cultivos"], "CultivosMunicipioResponse")
    assert payload_name["codigo_dane"] == codigo_dane, "❌ codigo_dane no coincide en /cultivos"
    assert payload_name["municipio"] == municipio, "❌ municipio no coincide en /cultivos"
    assert payload_name["cultivos"], "❌ La lista de cultivos está vacía"
    assert len(payload_name["cultivos"]) == len(set(payload_name["cultivos"])), "❌ Hay cultivos duplicados"
    assert all(cultivo in CULTIVOS_MVP for cultivo in payload_name["cultivos"]), (
        f"❌ Cultivos fuera del MVP: {payload_name['cultivos']}"
    )
    assert payload_name["cultivos"] == [c for c in CULTIVOS_MVP if c in payload_name["cultivos"]], (
        "❌ El orden de cultivos no respeta el orden canónico del MVP"
    )

    invalid_municipio = client.get(f"/cultivos/{quote('Municipio Inexistente', safe='')}")
    assert invalid_municipio.status_code == 404, (
        f"❌ Municipio inválido debería devolver 404 y respondió {invalid_municipio.status_code}"
    )

    # --- POST /predecir ---
    predict_resp = client.post(
        "/predecir",
        json={"municipio": municipio, "cultivo": cultivo_valido, "año": future_year},
    )
    assert predict_resp.status_code == 200, (
        f"❌ /predecir respondió {predict_resp.status_code}"
    )
    prediction = predict_resp.json()
    _assert_keys_exact(
        prediction,
        ["codigo_dane", "municipio", "departamento", "cultivo", "año", "rendimiento_esperado", "prob_riesgo_alto", "etiqueta_riesgo"],
        "PrediccionResponse",
    )
    assert prediction["codigo_dane"] == codigo_dane, "❌ codigo_dane no coincide en /predecir"
    assert prediction["municipio"] == municipio, "❌ municipio no coincide en /predecir"
    assert prediction["cultivo"] == cultivo_valido, "❌ cultivo no coincide en /predecir"
    assert prediction["año"] == future_year, "❌ año no coincide en /predecir"
    assert isinstance(prediction["rendimiento_esperado"], (int, float)) and math.isfinite(prediction["rendimiento_esperado"]), (
        f"❌ rendimiento_esperado inválido: {prediction['rendimiento_esperado']}"
    )
    assert isinstance(prediction["prob_riesgo_alto"], (int, float)) and math.isfinite(prediction["prob_riesgo_alto"]), (
        f"❌ prob_riesgo_alto inválido: {prediction['prob_riesgo_alto']}"
    )
    assert 0.0 <= float(prediction["prob_riesgo_alto"]) <= 1.0, (
        f"❌ prob_riesgo_alto fuera de rango: {prediction['prob_riesgo_alto']}"
    )
    assert prediction["etiqueta_riesgo"] in {"Bajo", "Medio", "Alto"}, (
        f"❌ etiqueta_riesgo inválida: {prediction['etiqueta_riesgo']}"
    )

    invalid_crop = client.post(
        "/predecir",
        json={"municipio": municipio, "cultivo": "Papa", "año": future_year},
    )
    assert invalid_crop.status_code == 422, (
        f"❌ Cultivo inválido debería devolver 422 y respondió {invalid_crop.status_code}"
    )

    invalid_year = client.post(
        "/predecir",
        json={"municipio": municipio, "cultivo": cultivo_valido, "año": future_year - 1},
    )
    assert invalid_year.status_code == 422, (
        f"❌ Año inválido debería devolver 422 y respondió {invalid_year.status_code}"
    )

    invalid_municipio_predict = client.post(
        "/predecir",
        json={"municipio": "Municipio Inexistente", "cultivo": cultivo_valido, "año": future_year},
    )
    assert invalid_municipio_predict.status_code == 404, (
        f"❌ Municipio inválido debería devolver 404 y respondió {invalid_municipio_predict.status_code}"
    )

    print("✅ Todos los criterios de aceptación pasaron")
    print("\nResumen:")
    print(f"  Municipios: {len(municipios)}/15")
    print(f"  Caso válido: {municipio} / {cultivo_valido} / {future_year}")
    print(f"  Rendimiento esperado: {prediction['rendimiento_esperado']:.4f}")
    print(f"  Prob. riesgo alto: {float(prediction['prob_riesgo_alto']):.4f}")
    print(f"  Etiqueta de riesgo: {prediction['etiqueta_riesgo']}")


if __name__ == "__main__":
    run_validations()
```

## Manejo de errores

| Situación                                                  | Comportamiento                         |
| ---------------------------------------------------------- | -------------------------------------- |
| Falta `feature_matrix.parquet` o cualquier modelo/meta     | `FileNotFoundError` con ruta explícita |
| `orchestrator.main` no existe o no exporta `app`           | `ImportError` o `ModuleNotFoundError`  |
| El endpoint responde con un status distinto al esperado    | `AssertionError` con prefijo `❌`      |
| El payload tiene claves faltantes, extra o tipos inválidos | `AssertionError` con prefijo `❌`      |
| `prob_riesgo_alto` queda fuera de rango                    | `AssertionError` con prefijo `❌`      |
