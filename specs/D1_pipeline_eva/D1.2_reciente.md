## Lógica de transformación
```
1. Llamar fetch_all(dataset_id='uejq-wxrr', where=<filtro SoQL>)
2. Convertir lista de dicts → pd.DataFrame
3. Renombrar columnas según mapeo
4. → _limpiar_eva(df, codigos_objetivo, fuente='reciente', prefijo_log='[D1.2]')
   (función privada compartida con D1.1 — ver implementación)
   4a. normalize_dane_code(codigo_dane) → str 5 dígitos (por consistencia)
   4b. Filtrar: conservar solo filas donde codigo_dane esté en codigos_objetivo
   4c. normalize_cultivo(cultivo) → 'Café'|'Cacao'|'Maíz'|None; descartar None
   4d. municipio y departamento: normalize_title_case() (ya vienen en Title Case,
       se aplica por consistencia y para hacer strip())
   4e. año = pd.to_numeric(año, errors='coerce').astype('Int64')
   4f. rendimiento = pd.to_numeric(rendimiento, errors='coerce')
       rendimiento donde valor == 0.0 → None
       rendimiento fuera de RENDIMIENTO_RANGOS[cultivo] → None
   4g. area_sembrada, area_cosechada, produccion = pd.to_numeric(..., errors='coerce')
   4h. ciclo: conservar 'PERMANENTE'|'TRANSITORIO'; cualquier otro valor → None
   4i. fuente = 'reciente'
   4j. Eliminar duplicados por (codigo_dane, cultivo, año, periodo, fuente)
   4k. Loggear warning por cada municipio objetivo sin datos
   4l. Retornar DataFrame con columnas en orden del esquema unificado
```

## Notas de implementación

### Función privada `_limpiar_eva`
La lógica de limpieza es idéntica entre D1.1 y D1.2. En lugar de duplicar ~80 líneas, se extrajo a `_limpiar_eva(df, codigos_objetivo, fuente, prefijo_log)`. Cada función pública solo hace la descarga y el renombrado de columnas (que sí difieren entre datasets), luego delega a `_limpiar_eva`.

### Municipio y departamento
Ya vienen en Title Case en este dataset. Se aplica `normalize_title_case()` de todas formas para garantizar `strip()` y consistencia con D1.1.

### Tipo de `año`
`Int64` (nullable integer) por las mismas razones que D1.1 — robustez ante valores nulos inesperados.

| Cultivo no reconocido | Descartar fila; loggear valor inesperado |

## Interfaz de la función

```python
def download_eva_reciente(
    codigos_dane: list[str] | None = None,
) -> pd.DataFrame:
    """
    Descarga y limpia EVA reciente 2019-2024.

    Args:
        codigos_dane: Lista de códigos DANE a filtrar.
                      Si None, usa MVP_CODIGOS de shared/dane_codes.py.

    Returns:
        DataFrame con esquema unificado del proyecto.
        Columnas: codigo_dane, municipio, departamento, cultivo,
                  año, periodo, rendimiento, area_sembrada,
                  area_cosechada, produccion, ciclo, fuente.

    Raises:
        ValueError: Si el resultado está vacío tras aplicar filtros.
        requests.RequestException: Si la API falla tras reintentos.
    """
```
